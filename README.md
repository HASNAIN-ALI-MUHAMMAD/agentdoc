# Agent Doc

## What is it?
A agentic document reader, that allows using best AI models for understanding documents as you go.

## Why is it?
When reading books or papers one must repeatedly shift focus from reader to browser and so on...
Agent doc incorporates agent within the reader, agent that have all the context about what you are reading and is ready to help you!

## Built with
Wails and Go
